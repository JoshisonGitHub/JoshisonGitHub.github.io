Rules

1. Must be an Anisoc member.

2. Don't vote for the same show twice.

3. When voting for a show that has already been voted for once, write the name of the show the exact same way.

4. Try not to vote for a show that we have already watched.

Information

1. A code is needed for your vote to count. This code can be found pinned in the members channel on the Anisoc Discord.

2. Vote for as many shows/films as you like!

3. Voting is anonymous.

4. See archive for shows that have already been watched.

5. Voting List will be updated once every week, so if your show hasn't been listed yet on the page fret not!
